# Bilbo Chatbot

### In use
- Slack API / Hubot
- Keras / tensorflow

### Installation
pip install -r requirements.txt

### Train
choose .txt
```
python train.py
```
### Generate text
set chars for sentence
```
python generate.py
```

### Set your hubot on Slack
get the API key
```
python discover.py
```
export HUBOT_SLACK_NAME, HUBOT_SLACK_TOKEN, HUBOT_SLACK_ID
